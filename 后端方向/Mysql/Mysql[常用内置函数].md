## 数学函数

- ceil 向上取整
- floor 向下取整
- round 四舍五入  round(1.21313,5)
- truncate 小数按精度截取 truncate(2.32432,2) 不会进行舍入操作，而是直接截断
- mod 求余数 mod(3,2) => 1
- abs绝对值
- power幂运算
- pi圆周率
- rand随机数 如果round传入了参数，则相当于对参数签名，参数一致时，返回值一样 
- sign取得数字符号 sign(-12) => -1  sign(12) => 1  sign(0) => 0 
- wxp 计算自然对数e的x方
