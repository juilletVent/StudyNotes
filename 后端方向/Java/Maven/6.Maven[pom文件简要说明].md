# pom文件简要说明

- dependencies 包含多个 dependencie
- dependency 定义单个包依赖：包含groupid artifactid version来确定唯一的包
- properties 定义属性集，包含自定一的xml元素，xml元素节点就是变量名称，内容就是变量值
- 设置java等级 主要是用了build标签
