## add2 和弦

构成：在普通三和弦基础上增加一个二度音即可

三和弦由两个三度音构成，此时在第一个三度音中间增加一个二度音，就形成了加 2 和弦：1235

这种和弦与原本的和弦功能基本一致；由于增加的二度音与原本的和弦根音和三度音比较近，给人一种模糊的感觉，稍微有那么一点儿不和弦，现代流行乐大量运用了加 2 和弦，流行音乐中几乎大多数三和弦都可以替换成加 2 和弦

```
C => Cadd2
Cm => Cmadd2
Em => Emadd2
bF => bFadd2
```
