## 什么是全音，什么是半音

全音伴音是描述的两个音之间的关系，如果两个音高之间存在另外一个唯一的音高，则这两个音之间的关系称之为一个全音，如果不存在任何其他音高，则称之为半音，如果存在不止一个其它音高，则是另外的名字（曾二度、大三度等），此处暂时不涉及：

```bash
# 全音
C => D => E
F => G => A
# 半音
E => F => #F
B => C => #C
```
