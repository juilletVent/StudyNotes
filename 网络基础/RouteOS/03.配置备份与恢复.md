## 配置备份与恢复

1. 备份配置

```shell
# system backup save name=<备份文件名称> [password=<备份密码>]
system backup save name=backup_20260306
system backup save name=backup_20260306 password=123456
```

2. 恢复配置

```shell
# system backup load name=<备份文件名称> [password=<备份密码>]
system backup load name=backup_20260306
system backup load name=backup_20260306 password=123456
```
