# 通网配置

RouteOS 虽好，但是配置起来还是比较复杂的，跟着下面的步骤来配置一下网络吧

## Interface 配置

菜单：Interface -> Interface（选项卡）

方便识别接口，修改接口接口名称，对应`ether1`、`ether2`接口修改名称为`lan`、`wan`,确认是对应的接口，不要搞错了

## Bridge 配置

菜单：Bridge -> Bridge（选项卡）

1. 列表中检查是否存在 `bridge` 接口，如果没有，点击 `+/new` 号添加一个 `bridge` 接口，确认保存
2. 切换选项卡，到 `Ports` 选项卡，将 `lan` 接口添加到 `bridge1` 中，确认保存

## IP 地址配置

菜单：IP -> Address（选项卡）

1. 点击 `+/new` 号添加一个IP地址，配置如下，然后保存：

```
address=192.168.1.1/24
netmask=192.168.1.0
interface=bridge1
```

2. 切换到 `Pool` 选项卡，添加DHCP地址空间，新建一个地址空间，address填入：`192.168.1.100-192.168.1.255`，名称填入：`dhcp-pool`，确认保存

## DHCP 配置

菜单：IP -> DHCP Server（选项卡）

点击 右上角`DHCP Setup` 按钮，跟随引导，配置DHCP服务器，确认保存

菜单：IP -> UPnP（选项卡）

勾选`Enabled`，确认保存

## PPPOE 配置

菜单：PPP

1. 点击 `+/new` 号添加一个PPPOE客户端
2. interface选择`wan`接口，然后切换到`Dial-out`选项卡，用户名和密码根据实际情况填写，确认保存
3. 观察 status 选项卡，等待PPPOE连接成功，成功后会自动获取到IP地址，确认保存

## Routes 配置检查

菜单：IP -> Routes（选项卡）

这个项目就是当前系统的路由表，检查相关内外网路由是否正确，如果缺少内网路由，或者路由表重复，酌情修改，路由不正确也会导致外网不通或者访问缓慢。

至此，通网配置基本完成。
