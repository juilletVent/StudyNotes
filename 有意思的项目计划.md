## 视频直播项目

项目目的：

1. 了解视频直播推流的基本原理
2. 熟悉前端视频流处理的基本操作、播放等 API
3. 统计视频直播的相关数据，主要是码率与流量间的关系

效益：如果能够实现，可以作为一个基础的视频直播项目向公司推荐立项，取得专项费用，用于支撑业务开发
