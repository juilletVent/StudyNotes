## 常用求导法则

**Tips：所有的求导法则，全都是通过导数定义进行求解的**

- $f(x) = C$ 则 $f(x)^{'} = 0$
- $f(x) = x^n$ 则 $f(x)^{'} = n * x^{n-1}$
- $f(x) = \sin x$ 则 $f(x)^{'} = \cos x$
- $f(x) = \cos x$ 则 $f(x)^{'} = -\sin x$
- $f(x) = a^x$ 则 $f(x)^{'} = a^x\ln a$
- $f(x) = e^x$ 则 $f(x)^{'} = e^x$
- $f(x) = \log_a x$ 则 $f(x)^{'} = \frac{1}{x\ln a}$
- $f(x) = \ln x$ 则 $f(x)^{'} = \frac{1}{x}$
