## if

与 shell 一样，bat 脚本支持反向验证（使用NOT完成：IF NOT condition）,不区分类型，区分大小写

```bat
@echo off
set a=1
set /A b=2

if %a% == %b% (
   echo equa
   echo equa2
)
```

## if/else

```bat
@echo off
set a=1
set /A b=2

if %a% == %b% (
  echo equal
) else (
  echo not equal
)
```

## 判断文件是否存在

```bat
@echo off
:: 判断当前文件夹下是否存在test.bat文件
if exist %~dp0test.bat (
   echo exists
) else (
   echo not exists
)
```

## 判断前一个程序退出的返回值

```bat
@echo off
route print
if %errorlevel%==0 (
   echo normal
) else (
   echo not normal
)
```

## 关于运算符

bat中的有常规的加减乘除求余运算符，位运算符，赋值运算符；没有 && 以及 ||这类逻辑表达式连接运算符
