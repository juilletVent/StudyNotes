## 方法论

- 通用的领域知识，一旦你下决心从事这个领域，能够尽早学习比较好，如果能一边学习，一边通过实践来打磨，就会掌握得更快。
- 专用领域知识，不一定要很早去学，有一个技巧是，当你用到的时候再去学习它们。
