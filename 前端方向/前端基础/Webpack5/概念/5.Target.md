## 模块解析

webpack 打包的代码可以在各种平台运行，因此需要指定打包目标平台,在代码中会应用相关的基础环境以及 API，如果设置错误会导致基础环境 API 找不到等问题

```js
module.exports = {
  // Nodejs
  target: "node",
  // Browser（default）
  target: "web",
  // webworker
  target: "webworker",
  // 编译为指定的目标平台代码，es6 es7等
  target: "es5",
  // electron主进程
  target: "electron-main",
  // ...
};
```
