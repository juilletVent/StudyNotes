## 代理捕获器与反射方法

代理可以捕获 13 种不同的基本操作。这些操作有各自不同的反射 API 方法、参数、 关联 ECMAScript 操作和不变式。

## 代理捕获器可用 API

- get
- set
- has
- defineProperty
- getOwnPropertyDescriptor
- deleteProperty
- ownKeys
- getPrototypeOf
- setPrototypeOf
- isExtensible
- preventExtensions
- apply
- construct
