## 简述

DOM1（ DOM Level 1）主要定义了 HTML 和 XML 文档的底层结构。 DOM2（ DOM Level 2）和 DOM3（ DOM Level 3）在这些结构之上加入更多交互能力，提供了更高级的 XML 特性。
