## 一般设计软件

一般设计软件的角度定义是，0 度指向屏幕右侧，正角度为逆时针（90deg 的时候，指向屏幕正上方）

## CSS 中

CSS 中 0 度指向屏幕正上方，正角度为顺时针（90deg 的时候，指向屏幕右方）

_Tips:在应用于渐变、变换等操作时，请特别注意，CSS 与常规的设计软件见的角度系统区别_
