## Keyof

使用keyof操作符，可以获取对象中的所有属性名，并返回一个联合类型：

```ts
type Point = { x: number; y: number };
type P = keyof Point;
// P = "x" | "y"
```

需要注意的高级场合：

```ts
// 搭配索引类型使用，数组：
type Arrayish = { [n: number]: unknown };
type A = keyof Arrayish;

// 边界特殊情况情况：对象
// 由于对象在使用数字进行索引时：obj[0] 等价与 obj["0"] 所以这里返回的签名是 string | number，必须知道这个特性，否则无法解释为什么是 string | number
type Mapish = { [k: string]: boolean };
type M = keyof Mapish;
// M = string | number
```

_Tips：keyof 一般不对基础联合类型使用，基本没什么意义_

_Tips：对于联合类型使用时，会提取所有联合类型的公共属性作（交集）为联合类型返回_

## Typeof

用于返回目标对象的类型：

```ts
// 对于一般类型的一般用法，意义不大
type Basic = typeof 0;
let point = { x: 10, y: 20 };
type Point = typeof point;

// 核心用法是搭配TS内的其他类型使用，比如：
function f() {
  return { x: 10, y: 3 };
}

type P = ReturnType<typeof f>;
// P = { x: number; y: number; }
```

这就很有用了，不用单独自己定义类型了，并且未来f变更了，类型也会自动重新推导，舒服~
