## never

never类型表示永远不会出现的类型，任何类型都不能赋值给never类型

搭配穷尽性检查，非常好用：

```typescript
// 第一版
type Shape = Circle | Square;

function getArea(shape: Shape) {
  switch (shape.kind) {
    case "circle":
      return Math.PI * shape.radius ** 2;
    case "square":
      return shape.sideLength ** 2;
    default:
      // 此处使用了never类型来进行穷尽性检查，如果有新的类型加入Shape联合类型中但是没有被处理，编译器会提示错误：
      // Type 'Triangle' is not assignable to type 'never'.
      // 因为任何类型都不可以赋值给never类型
      const _exhaustiveCheck: never = shape;
      return _exhaustiveCheck;
  }
}
```

## unknown

unknown类型表示未知类型，任何类型都可以赋值给unknown类型，如果你不确定一个变量的类型时，最好使用unknown类型，而不是any类型，因为any类型会丢失类型信息
