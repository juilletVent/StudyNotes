## 联合类型分发（拆分）

联合体字面量类型将会在插值位置进行分发：

```ts
type EmailLocaleIDs = "welcome_email" | "email_heading";
type FooterLocaleIDs = "footer_title" | "footer_sendoff";

type AllLocaleIDs = `${EmailLocaleIDs | FooterLocaleIDs}_id`; // 总计生成四个类型组成的联合类型
```

_Tips：通常建议对大型字符串联合体使用预先生成，但这在较小的情况下也很有用_

## 使用字符串模板加联合类型分发创建函数签名

```ts
const person = {
  firstName: "Saoirse",
  lastName: "Ronan",
  age: 26,
};

type Person = typeof person;

type PropEventSource<Type> = {
  on(
    // 模板字符串限定修改提取相关字面量并创建相关事件名
    event: `${keyof Type & string}Changed`,
    callback: (newVal: any) => void,
  ): void;
};

declare function makeWatchedObject<Type>(
  obj: Type,
): Type & PropEventSource<Type>;

const personListener = makeWatchedObject(person);

personListener.on("firstNameChanged", (nVal) => {}); // ok, 但是nVal为any
personListener.on("ageChanged", (nVal) => {}); // ok, 但是nVal为any
personListener.on("ageChange", (nVal) => {}); // error
```

更进一步，使用泛型加条件类型进一步限制回调函数的参数类型，而不是使用any：

```ts
const person = {
  firstName: "Saoirse",
  lastName: "Ronan",
  age: 26,
};

type Person = typeof person;

type PropEventSource<T> = {
  // 函数写法
  // on<K extends string & keyof T>(eventName: `${string & K}Changed`, callback: (newValue: T[K]) => void): void;
  // 属性写法
  on: <K extends string & keyof T>(
    event: `${K}Changed`,
    callback: (nVal: T[K]) => void,
  ) => void;
};
declare function makeWatchedObject<Type>(
  obj: Type,
): Type & PropEventSource<Type>;

const personListener = makeWatchedObject(person);

personListener.on("firstNameChanged", (nVal) => {});
personListener.on("ageChanged", (nVal) => {});
personListener.on("ageChange", (nVal) => {});
```

带实现的完整案例：

```ts
const person = {
  firstName: "Saoirse",
  lastName: "Ronan",
  age: 26,
};

type Person = typeof person;

type PropEventSource<T> = {
  // 函数写法
  // on<K extends string & keyof T>(eventName: `${string & K}Changed`, callback: (newValue: T[K]) => void): void;
  // 属性写法
  on: <K extends string & keyof T>(
    event: `${K}Changed`,
    callback: (nVal: T[K]) => void,
  ) => void;
};

// 创建裁剪类型用于处理Changed尾巴
type RemoveChanged<T extends string> = T extends `${infer N}Changed`
  ? N
  : never;

function makeWatchedObject<Type>(obj: Type): Type & PropEventSource<Type> {
  return {
    ...obj,
    on: (event, callback) => {
      // 裁掉尾巴
      const key = event.slice(0, -"Changed".length);
      console.log(
        "%s => %s",
        event,
        // 使用工具类型处理属性联合类型
        callback(obj[key as RemoveChanged<typeof key>]),
      );
    },
  };
}

const personListener = makeWatchedObject(person);

personListener.on("firstNameChanged", (nVal) => {
  return `${nVal} AHA~`;
});
personListener.on("ageChanged", (nVal) => {
  return nVal + 10086;
});

personListener.on("ageChange", (nVal) => {}); // error
```

## 一些与字符串字面量类型有关的工具函数

- `Uppercase<StringType>`：将字符串中的每个字符转换为大写形式
- `Lowercase<StringType>`：将字符串中的每个字符转换为小写形式
- `Capitalize<StringType>`：将字符串中的第一个字符转换为大写形式
- `Uncapitalize<StringType>`：将字符串中的第一个字符转换为小写形式
