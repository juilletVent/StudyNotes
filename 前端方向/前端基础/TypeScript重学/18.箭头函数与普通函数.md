| 特性              | 普通函数                   | 箭头函数                          |
| ----------------- | -------------------------- | --------------------------------- |
| `this`            | 动态绑定（由调用方式决定） | 词法绑定（来自外层作用域）        |
| `arguments`       | 有自己的                   | 没有，继承外层                    |
| `prototype`       | 有                         | 没有                              |
| `new`             | 可以                       | 不可以                            |
| `call/apply/bind` | 可以改变 `this`            | 无法改变 `this`                   |
| `yield`           | Generator 可用             | 不支持                            |
| `super`           | 在方法中可用               | 继承外层方法的 `super`            |
| `new.target`      | 有自己的                   | 继承外层                          |
| 适合做对象方法    | ✔️                         | 一般不推荐（除非需要固定 `this`） |
| 适合做回调        | 可以，但容易丢失 `this`    | ✔️ 非常适合                       |
