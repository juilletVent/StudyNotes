## 模块定义

任何包含 `import` 或 `export` 的文件都被是否一个模块。

相反，没有任何顶级导入或导出声明的文件将被视为脚本，其内容可在全局范围内使用。

## 导入、导出类型

```ts
export type Cat = { breed: string; yearOfBirth: number };
export type Dog = { breeds: string[]; yearOfBirth: number };
type Animal = Cat | Dog;
export type { Animal };

// 使用
import type { Animal } from "./animal";
// or
import { type Animal } from "./animal";
```
