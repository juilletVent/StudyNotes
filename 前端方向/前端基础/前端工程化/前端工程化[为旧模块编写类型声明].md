## 为老旧的库编写类型声明

1. 项目根目录下新建目录 typings
2. 新建与库同名的 xxxx.d.ts 类型声明文件
3. 在 tsconfig.json 中配置 compilerOptions.paths 选项：

```json
"paths": {
  "*": ["node_modules/@types", "./typings/*"],
  // ...
}
```
